# RE9-AdvancedFOV
specifically made for RE9 speedrunning, customization heavily stripped

how to use:
1. make sure you don't have any other FOV mods installed
2. [reframework](https://github.com/praydog/REFramework-nightly/releases) needs to be installed to deal with capcom's integrity checks
3. take the FOV display with a pinch of salt
4. run the program and it will hook & load automatically
5. you can close it after it loads
6. you're good

troubleshooting:
1. if you get HookMemory: Unknown Error, you're most likely missing vcredist, [install this ](https://www.techpowerup.com/download/visual-c-redistributable-runtime-package-all-in-one/)
2. if you get one of the ScanMemory errors, make sure you're not heavily downpatched and try removing all the other mods, one of them might be getting in a way, don't remove reframework itself
3. your antivirus might give a false alert and remove some of the important files or partially stop the program, try excluding the whole folder from scanning if you're having issues
